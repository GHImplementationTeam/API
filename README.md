# API [![Build Status](https://travis-ci.org/GHImplementationTeam/API.svg?branch=master)](https://travis-ci.org/GHImplementationTeam/API)
API backing frontend and user-facing API endpoints

Installation
------------
```
npm install -g sails
npm install
```

Run for development
-------------------
```
sails lift
```

Run for production
------------------
```
sails lift --prod
```
